Тег
Тег — это специальный символ разметки, который применяется для вставки различных элементов на веб-страницу таких как: рисунки, таблицы, ссылки и др., и для изменения их вида.
HTML-документ
HTML-документ - это обычный текстовый файл, который может содержать в себе текст, теги и стили. Изображения и другие объекты хранятся отдельно. Содержимое такого файла обычно называется HTML-код.
Сайт
Cайт — это набор отдельных веб-страниц, которые связаны между собой ссылками и единым оформлением.
Язык программирования
Язык программирования - это формальный язык, используемый для формулирования структур данных и алгоритмов, то есть вычислительных правил которые могут быть выполнены компьютером.
JavaScript
JavaScript - это мультипарадигменный язык программирования.
